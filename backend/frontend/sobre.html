Sobre o Scrutiny

Análise técnica automática de perfis GitHub com inteligência artificial.

💡 Origem

O Scrutiny nasceu da necessidade de analisar perfis de desenvolvedores de forma rápida e objetiva. Como o GitHub é o portfólio real de um dev, a ferramenta lê esses dados públicos e gera uma análise técnica detalhada com o apoio de IA — em segundos.

🎯 Para quem é

Recrutadores que precisam avaliar candidatos tecnicamente de forma rápida, sem precisar ler repositório por repositório. E desenvolvedores que querem entender seus pontos fortes e o que melhorar no próprio portfólio.

⚙️ Stack

Backend Node.js + Express
IA Groq API · LLaMA 3.3 70B
Dados GitHub API (Octokit)
Frontend HTML, CSS, JavaScript
