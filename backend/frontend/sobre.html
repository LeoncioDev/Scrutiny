Sobre o Scrutiny

Analisa perfis GitHub e gera uma opinião honesta sobre os projetos — tecnologias detectadas, documentação e o que chama atenção.

💡 Origem

O Scrutiny nasceu da necessidade de entender um perfil GitHub de forma rápida e objetiva. Em vez de abrir repositório por repositório, a ferramenta lê os READMEs e detecta as tecnologias automaticamente — e usa IA para dar uma opinião direta sobre o que encontrou.

🎯 Para quem é

Para qualquer pessoa que queira uma leitura rápida de um perfil — seja um dev que quer saber como o próprio perfil aparece para quem visita, ou alguém que quer entender o que os projetos de outra pessoa dizem sobre ela.

⚙️ Stack

Backend Node.js · Express
IA Groq API · LLaMA 3.3 70B
Dados GitHub API (Octokit)
Frontend HTML · CSS · JavaScript
